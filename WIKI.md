# 📖 Shadow Ebook 使用指南

## 功能概览

Shadow Ebook 是一款免费开源的英语电子书阅读与跟读学习工具，支持 EPUB 导入、点击查词、TTS 朗读、蓝思值计算，专为英语学习者设计。

---

## 📱 快速开始

### 安装依赖
```bash
pip install flask edge-tts
```

### 运行
```bash
python tutor_web.py
```
打开浏览器访问：**http://localhost:5002**

---

## 📚 电子书阅读 (/)

### 导入书籍
1. 点击 **➕ 导入新书** 选择 EPUB 文件
2. 等待解析完成
3. 选择章节开始阅读

### 核心功能
- **点击查词**：点击任意英文单词，即时显示中英文释义
- **TTS朗读**：点击句子自动朗读（Microsoft edge-tts）
- **生词本**：点击单词可加入生词本，浏览器本地保存
- **蓝思值**：实时计算当前书籍的阅读等级 (AR值)
- **跟读评分**：切换到跟读模式，录音对比原音

---

## 🗣️ 跟读学习 (/tutor)

### 学习内容
- **新概念英语青少版 2A**：9个单元，涵盖核心语法
- **Magic Tree House #29**：圣诞奇案故事跟读

### 跟读流程
1. 选择学习内容（书籍/单元）
2. 点击 **🎤 开始跟读** 录音
3. 录音完成后显示波形对比，可回听自己的发音
4. 完成跟读后做理解测验题

### 波形对比
录音后会显示：
- 🔵 蓝色波形：原音
- 🟢 绿色波形：你的录音

## 📖 语法学习 (/grammar)

### 支持级别
- **KET (A2)**：一般现在时、现在进行时、一般过去时、There be句型、be going to、情态动词、形容词比较级、频率表达、名词所有格、代词系统、介词用法、祈使句
- **PET (B1)**：现在完成时、过去完成时、被动语态、条件句、间接引语、定语从句、情态动词完成时、used to、动名词/不定式、数量词、状语从句、并列连词
- **新概念英语青少版 2A 核心语法**

### 使用方法
1. 选择 KET 或 PET 标签页
2. 点击任意语法卡片查看详细讲解
3. 点击 **🎯 开始练习** 做选择题测试
4. 答题反馈即时显示，正确率自动保存

### 卡片状态标识
- 无标识：未开始
- 🟡 黄点：进行中
- 🟢 绿勾：已掌握 (>80%正确率)

## 📊 学习统计 (/stats)

### 数据概览
- **🔥 连续打卡天数**：记录连续学习不间断天数
- **⏱️ 总学习时长**：累计学习时间
- **📚 学习词汇**：已收录的生词数量
- **🗣️ 跟读句子**：已跟读练习的句子数
- **✅ 掌握词汇**：通过艾宾浩斯复习已掌握的单词

### 学习时长图表
显示近7天每日学习时长，柱状图直观展示学习习惯。

### 掌握进度
- 📚 词汇掌握进度
- 📖 语法练习进度
- 🗣️ 跟读练习进度

## 🔔 艾宾浩斯复习

### 原理
根据遗忘曲线科学安排复习时间：
- 第1天 → 第3天 → 第7天 → 第14天 → 第30天

### 使用方法
1. 在电子书页面的**生词本**区域
2. 有待复习单词时显示红色提醒徽章
3. 点击 **📝 开始复习**
4. 认识点击绿色按钮，不认识点击红色按钮
5. 系统自动调整下次复习时间

## 📖 章节理解测验

### 触发方式
在电子书阅读器底部点击 **📖 章节测验** 按钮

### 题目类型
- 主旨大意题
- 细节理解题
- 词汇推断题
- 篇章结构题

### 完成后
显示正确率和答题反馈，帮助检验阅读理解效果

---

## 📚 词汇量测试与书单推荐

### 测试入口
统计页面 (/stats) → 点击 **开始测试**

### 测试内容
10道选择题，测试词汇理解能力

### 等级划分
| 正确率 | 等级 | 词汇量 |
|--------|------|--------|
| ≥90% | 高级 | 3000+ |
| ≥70% | 中高级 | 2000-3000 |
| ≥50% | 中级 | 1000-2000 |
| <50% | 初级 | 500-1000 |

### 书单推荐
根据测试结果推荐适合当前水平的书籍，从入门到高级循序渐进。

## 📱 PWA 离线支持

### 安装到桌面
1. 在 Chrome/Safari 中打开网站
2. 点击地址栏右侧的安装图标
3. 确认添加到主屏幕

### 离线可用功能
- 已缓存的页面浏览
- 离线查看已加载的书籍内容
- 学习数据自动保存

## 💾 数据存储

所有学习数据保存在浏览器 localStorage 中：

| Key | 内容 |
|-----|------|
| `vocabulary` | 生词本（包含艾宾浩斯复习数据）|
| `shadowStats` | 学习统计（时长、打卡、进度等）|
| `grammarProgress` | 语法练习进度 |
| `books` | 已导入的书籍数据 |

如需重置，运行 `localStorage.clear()` 或在统计页面点击"重置所有数据"。
