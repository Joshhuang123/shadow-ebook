# 🦊 Shadow Learning 安装使用指南

> 适用于小学生原版阅读社团

---

## 一、需要的设备

| 设备 | 用途 | 说明 |
|------|------|------|
| 老师电脑 | 运行服务器 | Mac/Windows 均可，需联网 |
| 学生平板/电脑 | 访问学习 | iPad/Android平板/笔记本电脑，浏览器访问 |

---

## 二、安装步骤

### 第一步：下载项目

点击老师电脑上的 **Code** 按钮 → **Download ZIP**

或者终端运行：
```bash
git clone https://github.com/你的用户名/shadow-learning.git
cd shadow-learning
```

### 第二步：安装依赖

#### Mac 用户
```bash
./setup.sh
```

#### Windows 用户
```bash
# 打开 PowerShell
py -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 第三步：启动服务

```bash
# 激活虚拟环境
source venv/bin/activate   # Mac
# venv\Scripts\activate    # Windows

# 启动
python3 tutor_web.py
```

启动成功后会显示：
```
🦊 Shadow Learning - 原版阅读社团版
======================================
🌐 本机访问: http://localhost:5002
📱 局域网访问: http://192.168.1.x:5002
```

---

## 三、学生使用

### 3.1 连接学习系统

1. 学生平板连接**和老师电脑同一个 WiFi**
2. 打开浏览器（Safari 或 Chrome）
3. 地址栏输入老师电脑上显示的 **局域网地址**（如 `http://192.168.1.100:5002`）

### 3.2 功能介绍

| 功能 | 位置 | 说明 |
|------|------|------|
| 📖 电子书阅读 | 首页 | 阅读英文原著，查词，加生词本 |
| 🎧 跟读练习 | 跟读学习 | 听标准发音，录音练习，波形对比 |
| 📝 语法学习 | 语法学习 | 语法讲解和练习 |
| 📊 学习统计 | 右上角 | 查看学习时长、词汇量等 |

### 3.3 跟读练习步骤

1. 选择学习内容（新概念/Magic Tree House）
2. 点击**听标准发音**听原音
3. 点击**开始跟读**录音
4. 听自己的录音，对比波形
5. 做理解测验题

### 3.4 生词复习（艾宾浩斯）

1. 阅读时点击不认识的单词
2. 点击 **+ 生词本** 收藏
3. 第二天回来，系统会提醒复习
4. 复习后根据记忆曲线自动安排下次复习时间

## 四、添加新书

1. 打开 **电子书** 页面
2. 点击 **导入 EPUB** 按钮
3. 选择本地 EPUB 文件
4. 等待解析完成（约10秒）
5. 新书出现在书架上

## 五、常见问题

### Q: 平板上无法录音？
**A:** 检查平板浏览器是否授予了**麦克风权限**。首次使用会弹出授权提示，请点击允许。

### Q: 学生打不开页面？
**A:**
1. 确认平板和老师在同一个 WiFi
2. 确认老师电脑的服务还在运行（终端不要关闭）
3. 尝试刷新页面或重启浏览器

### Q: 提示"无法访问麦克风"？
**A:** iOS Safari 需要用户主动点击录音按钮才会请求权限，不要让平板进入锁屏状态。

### Q: 声音播放有延迟？
**A:** 首次使用需要联网下载语音库，之后会缓存到本地。

### Q: 数据会丢失吗？
**A:** 数据保存在浏览器本地。老师可以建议学生每周打开一次"学习统计"页面截图保存，或定期导出。
